# 十五日で、日本語。｜15天学术日语 · 大学院受験(中国文学)· 东大应援版

写给未来的东大生的 15 天日语唤醒手册。对象:已过 JLPT N1 / 日语专八、多年不用、
正备考日本大学院(专业:中国文学)的人。

每天一页:一组带假名的学术词汇 + 三个书面语法,例句全部取材于中国文学史——
唐宋明清古文、元代戏曲、明清小说、评点文论、现当代研究著作与日本汉学。
背日语的同时复习文学史常识。学完一天盖一个红章(スタンプラリー),十五章集满上考场。

**在线阅读:** https://zhimeng.page/nihongo-15days/
